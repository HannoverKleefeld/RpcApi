Pre define API constants.

Pre defines the need Constants. Defines the need Constants and create if needed the directory structure

  • Set RPC_LIB_DIR to : Directory where rpcapi.php located
  • Set RPC_MESSAGES_DIR to : Directory where rpcmessages.en.inc located
    Default is RPC_LIB_DIR/messages
  • Set RPC_CONFIG_DIR to : Directory where API Json config files to saved and lod from
    Default is RPC_LIB_DIR/config
  • Set RPC_CONFIG_DIR to : Directory where import remote device files are cached
    Default is RPC_LIB_DIR/cache
