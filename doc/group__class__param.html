Structure of an function argument

Example

Get Param

$api = new RpcApi('demo_config.json', new RpcLogger());
$function = $api->GetFunction('GetVolume');
$arguments= $function->{FUNCTION_PARAMS};
$argument = $arguments->{PARAMS_IN}[0];

Structure

$name = $argument->{VALUE_NAME};
$type = $argument->{VALUE_TYPE};
$default = $argument->{VALUE_DEFAULT};
$description = $argument->{VALUE_DESC};
$min = $argument->{VALUE_MIN};
$max = $argument->{VALUE_MAX};
$step = $argument->{VALUE_STEP};
$list = $argument->{VALUE_LIST};
