RpcApi::GetFunctionNames (   $ServiceName = '',
  $IncludeServiceName = false 
)

Get list of registered functions.

if $IncludeServiceNames is true then returns a KeyValuePair array [ServiceName]=FunctionNames when not get a simple string array with FunctionNames ['name1','name2'...]
if no Param given then returns a list of all registered device functions.

Parameters
string$ServiceNameif set only functions from this service returned. Default=''
bool$IncludeServiceNameif set you get . Default=false
Returns
null|array null if no service or function found
