Detailed Description

Manage RpcApi Debug and Error Messages.

PHP Class to manage RpcApi Debug and Error Messages
allows the RpcApi to output errors and debug messages to the screen and the logfile
The protected method doOutput can be overridden to render all output from the screen itself
The public methods Debug and Error can be used for user-specific messages
