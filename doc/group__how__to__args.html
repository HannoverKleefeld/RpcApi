Use call arguments.

first we load the api

$api = new RpcApi('demo_config.json', new RpcLogger());
echo "Api created";

then we look at the RpcApi::Help for the command GetVolume

$api->Help('GetVolume');

returns
GetVolume (int $InstanceID = 0, string $Channel = "Master" [Master|LF|RF] ) => int CurrentVolume
you can see that $InstanceID has 0 and $Channel has Master as default value.
Now you can call method of 3 ways

  1. Normaly Since the arguments $InstanceID and $Channel have a default value, they do not need to be specified
    echo "\nVolume: ".$api->GetVolume();
    Note
    unless one of the following arguments should be handed over
  2. Order of the arguments
    $InstanceID = 0;
    $Channel = "Master";
    echo "\nVolume: ".$api->GetVolume($InstanceIDl);
    echo "\nVolume: ".$api->GetVolume($InstanceID,$Channel);
    Note
    This call, however, will generate an error because the InstanceID must be passed as the first argument, even if it contains a default value
    echo "\nVolume: ".$api->GetVolume($Channel);
  3. Named arguments In this type of call, the order of the parameters is not significant because they are passed as key names
    echo "\nVolume: ".$api->GetVolume(['Channel'=>$Channel]);
    // or
    echo "\nVolume: ".$api->GetVolume(['Channel'=>$Channel,'InstanceID'=>$InstanceID]);
