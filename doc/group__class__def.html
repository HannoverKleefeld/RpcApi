Example

Get Definition

$api = new RpcApi('demo_config.json', new RpcLogger());
$def = $api->GetModelDef();

Structure

$manufacturer= $def->{DEF_MANU};
$model = $def->{DEF_MODEL};
$options = $def->{DEF_OPTIONS};
$path = $def->{DEF_PATH};

Parameters
string$manufacturerrecognized devices manufacturer
string$modelrecognized devices model number
string$optionsnot used
string$pathpath to the device description file
string$propertysrecognized devices propertys
string$versionversion number to compare the version to be loaded and the current version
Attention
if the Api version number is not identical to the one in the file to be loaded then the loading process is aborted with an error
