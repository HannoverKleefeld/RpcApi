Device structure

$this->_device (array) of stdClass objects

Device config structure

$config = &$this->_device->{DEVICE_CONFIG} (stdClass)
$config->{CONFIG_SCHEME} (string) scheme 'http' or 'https'
$config->{CONFIG_HOST} (string) host without http(s)
$config->{CONFIG_PORT} (int) communication port
$config->{CONFIG_CONNTYPE} (int) once of
$config->{CONFIG_LOGIN_U} (string) username
$config->{CONFIG_LOGIN_P} (string) password

Device definition structure

$def = &$this->_device->{DEVICE_DEF} (stdClass)
$def->{DEF_MANU} (string) Detected device manufacturer
$def->{DEF_MODEL} (string) Detected device modelnumber
$def->{DEF_OPTIONS} (int) Device options
$def->{DEF_PROPS} (int) Device propertys
$def->{DEF_VERSION} (string) Api version

Device icons structure

$icons = &$this->_device{DEVICE_ICONS} (array)
foreach($icons as $icon){
$icon->{ICON_URL} (string) icon location
$icon->{ICON_HEIGHT}(int) icon height in px
$icon->{ICON_WIDTH} (int) icon widht in px
$icon->{ICON_DEPTH} (string) icon color depth
$icon->{ICON_MIME} (string) icon mime type ('jpg','png' ....)
}
