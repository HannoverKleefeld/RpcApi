Valid property options.

  • PROP_VOLUME_CONTROL the device can control the volume
  • PROP_MUTE_CONTROL the device can control the mute state
  • PROP_BALANCE_CONTROL the device can control the balance
  • PROP_TREBLE_CONTROL the device can control the treble value
  • PROP_BASS_CONTROL the device can control the bass value
  • PROP_LOUDNESS_CONTROL the device can control the loudness state
  • PROP_BRIGHTNESS_CONTROL the device can control the brightness
  • PROP_CONTRAST_CONTROL the device can control the contrast
  • PROP_SHARPNESS_CONTROL the device can control the sharpness
  • PROP_SOURCE_CONTROL the device can control the source input
  • PROP_PLAY_CONTROL the device can control the playstate PLAY,PAUSE,STOP...
  • PROP_REMOTE_CONTROL the device can emulate the remote control
  • PROP_CONTENT_BROWSER the device can browse the content
  • PROP_EVENTS the device supports events
  • PROP_PHONE the device supports phones
  • PROP_USER the device has user functions

Example

if($api->HasProps(PROP_EVENTS)){
$events = $api->RegisterEvents('','http://mycallbackurl');
}

or combined props

if($api->HasProps(PROP_VOLUME_CONTROL + PROP_BASS_CONNTROL)){
$volume = $api->GetVolume();
$bass = $api->GetBass();
}
