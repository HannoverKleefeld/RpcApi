Collaboration diagram for Predefined constants:

Overview of the avaible predefined constants.

RPC_LIB_DIR directory where rpcapi.php located
RPC_MESSAGES_DIR directory where rpcmessages.en.inc located
RPC_CONFIG_DIR directory where API json config files to saved and load from
RPC_CACHE_DIR directory where import remote device files are cached
RPC_EXTERNAL_DESCRIPTIONS if true then all descriptions are stored in the external file
RPC_LANGUAGE if not defined then default is 'en'


Example:

<?php
define('RPC_LANGUAGE','de');
define('RPC_CACHE_DIR','/my_cache_path');
define('RPC_CONFIG_DIR','/all_my_configs');
define('RPC_EXTERNAL_DESCRIPTIONS',false);
require_once 'rpcapi.php';
// or
require_once 'loader.php';
// then
$api = new RpcApi('demo_config.json', new RpcLogger());
// ....
