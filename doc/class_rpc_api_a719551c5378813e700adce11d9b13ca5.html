Get help for functions.

This method provides help information about all registered functions.

  • If a name is specified and this function exists then the result returns only to this information.
  • If no name is specified then all functions are listed
  • If # as name given then returns only API functions
    Parameters
    string$FunctionNamefunction name you need help with or empty for everyone
    int$HelpModeonce of HELP_SHORT (default), HELP_NORMAL or HELP_FULL
    boolean$ReturnResultif true method returns the result as string when not echo the result
    int$HelpWidhtthe max widht of created Help (only $HelpMode != HELP_SHORT)
    Returns
    string|null if $returnResult then return null
