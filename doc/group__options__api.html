Valid Api Options.

  • OPTION_DEFAULT_TO_END set params with a default to end of function params.
  • OPTION_CHECK_VALUES check if Params missed or failed before make a call
  • OPTION_RESULT_FILTER filter return results

Example

$api->SetApiOptions(OPTION_CHECK_VALUES + OPTION_RESULT_FILTER);
