Example Create language file

This is the content of the English language file, change the text into your language and save it as "rpcmessages.xx.inc".
The "xx" stands for the language abbreviation.
en = english
de = german
it = Italian
.....

To use this new file, there are two ways
  1. Define the Global Language constant in your Projet, before include RpcApi 
     define('RpcLanguage','en');
 require_once('rpcapi.php');
  2. Set the Language when creating a custom RpcMessage object see Create-Message and Example Create Logger
