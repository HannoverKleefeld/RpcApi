Uses of Logger.

demonstrates some possibilities to create a user defined logger

Example

  • Simple logger with default options
    $logger = new RpcLogger();
  • Extended Logger with own Translation Message object
    $mesages = new RpcMessage('en');
    $logger = new RpcLoger(DEBUG_ALL,'', $messages);
    // or
    $logger = new RpcLoger(DEBUG_CALL,'', $messages);
    Parameters
    string$messagesclass of Message
    See also
    Create Language File or Logger
    RpcLogger::SetMessage
  • Logger with logfile output
    $logger = new RpcLogger(DEBUG_WARN, 'logfile.log');
    // or
    $logger = new RpcLogger();
    $logger->SetLogFile('logfile.log');
    See also
    RpcLogger::SetLogFile
