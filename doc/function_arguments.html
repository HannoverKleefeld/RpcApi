Function arguments

Service function parameters can be passed in two ways

  1. order of the arguments as defined
    $instanceID=0;
    $volume=50;
    $api->SetVolume($instanceID,$volume);
  2. use the argument name
    $instanceID=0;
    $volume=50;
    // $instanceID is not needed because this is a default value defined
    $api->SetVolume(['DesiredVolume'=>$volume])
Zu beachten
if the Api options OPTION_DEFAULT_TO_END is set then the call is in the following way because all arguments with default values are put to the back
$instanceID=0;
$volume=50;
// $instanceID is not needed because this is a default value defined
$api->SetVolume($volume);
