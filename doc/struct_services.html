Service functions structure

$functions=$service->{SERVICE_FUNCTIONS};
foreach($functions as $functionName => $function){
$function->{FUNCTION_NAME} (string)Name of function
$function->{FUNCTION_PARAMS} (array) Function params in / out
$function->{FUNCTION_SOURCE} (string)PHP sourcecode if function injected
$function->{FUNCTION_EXPORT} (array) Variables to export in eval when called sourcecode
$function->{FUNCTION_DESC} (string)Description , only avaible if api injected
}

Function params structure

$params_in = $function->{PARAMS_IN}
$params_out = $function->{PARAMS_OUT}
foreach($params_in as $param){
$param->{VALUE_NAME} (string)Name of parameter
$param->{VALUE_TYPE} (int) Type of parameter
$param->{VALUE_MIN} (int) Min value (only set if found while importing)
$param->{VALUE_MAX} (int) Max value (only set if found while importing)
$param->{VALUE_STEP} (int) Step value (only set if found while importing)
$param->{VALUE_LIST} (array) List of allowed values (only set if found while importing)
$param->{VALUE_DESC} (string)Description of Parameter (only set when injecting while importing)
$param->{VALUE_DEFAULT}(mixed) Default value used by call functions when no value for this param given
}
$params_out same as $params_in
