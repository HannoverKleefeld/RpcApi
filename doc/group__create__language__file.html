Demonstrates creating own language file.

This is the content of the English language file, change the text into your language and save it as "rpcmessages.xx.inc" in subdir messages.
The "xx" stands for the language abbreviation.
en = english
de = german
it = Italian
.....

To use this new file, there are two ways

  1. Define the Global Language constant in your Projet, before include RpcApi
    define('RPC_LANGUAGE','en');
    require_once('rpcapi.php');
  2. Set the Language when creating a custom RpcMessage object
